# Description
This code challenge is a flask-based RESTful web application that manages heroes, their powers and the strength of each hero-power relationship. 

# Features
- View all heroes and individual heroes
- View all powers
- Update an existing power's description
- Assign powers to heroes
- Data validation

# Technologies used
- Python
- Flask
- Flask-SQLAlchemy
- SQLAlchemy
- SQLite

# Setup Instructions
1. Clone repository
   git clone <repo-url>
   cd <project-folder>
2. Install dependencies
   pipenv install
   pipenv shell
3. Set up the database
   Flask db init
   Flask db migrate
   Flask db upgrade head
4. Seed the database
   Python seed.py
5. Run the application
   flask run
